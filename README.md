# DailyTaskNotifier

## Descripción

Serverless DailyTaskNotifier. 

- Usamos **NestJS** con arquitectura modular.
- Aplicamos patrones **Provider** y **Singleton**.
- Seguimos principios **SOLID**.

## Instalación

```bash
$ npm install
```

## Ejecutar el App

```bash
# Local
$ sls offline

# Despliegue AWS (asegurar de tener configurado sus credenciales)
$ sls deploy
```

## Test

```bash
$ npm run test
```
